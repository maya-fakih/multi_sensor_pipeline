1st you should love robots and do it with pleasure or else really go find something else to do with your life :)

# create your project package

in case you dont have a ros2_ws directory create one using
cd
mkdir ros2_ws

now navigate to your working directory ros2_ws by
cd ~/ros2_ws
ros2 pkg create --build-type amment_cmake <package name>
cd your_package
